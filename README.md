# Who Is At Home

A simple pihole analyiser to alert on network arrivals and departures.

## Description

The purpose of this project is to provide a simple tool to alerts when people arrive, or leave home. This is done by monitoring DNS requests from a mobile phone or smart watch.  

## Getting Started

### Dependencies

* Linux, PiHole, Python3

### Installing

* Clone this repository locally

### Executing program

* Run the whosathome.py  
```
python3 whosathome.py
```
